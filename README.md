# Basic Calculator

This is a simple calculator program that supports addition, subtraction, multiplication, and division.

I build a menu base basic calculator where a user can

* add multiple numbers
* substract multiple numbers
* multiply multiple numbers
* divided multiple numbers

## def:

def is used to create new funtion. And here i use def to create sum, sbstract, multiply, division and a main function named calculator.

## print():

I use print function to display options for user. They can choose options according to their need.

## input():

I use input() function to take users choice. they can put choice according to displayed option.

## addition:

If user choose option 1  then ask for multiple integer numbers that seperate by space. then calculate the sum and print the final output. * is used before parameters to take multiple number as inputs.

## substraction:

If user choose option 2 then ask for numbers seperated by space. user can input multiple number to subtract from the first number she puts.

## multiplication:

### math:

I import math function to call math.prod() which is a built in function used to multiply multiple numbers.

If user choose option 3 then ask for multiple numbers seperated by space . They can put as many numbers they want to multiply.

## divition:

If user choose option 4 it asks for multiple input seperated by space. the inputed first number will be divided by the multiplication of the rest of the numbers they put.

## split():

split() function is used to seperate input by space.

## map():

I use map funtion to apply int() function to all the given input as map function is used to apply a function to each element of a list.

## while:

I use while loop as i don't no how many times an user would continue this calculations.

## break:

I use break function to stop the calculation. if the user choose option 5 then he can exit from the calculation.
